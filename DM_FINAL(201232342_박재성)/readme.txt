< 학습데이터와 평가데이터 생성 코드>

1. preprocessing1.py를 run하면 5000개의 단어 파일이 만들어 집니다.

2. preprocessing2.py를 run하면 5000개의 파일을 읽어 feature안에 Input_Data와 Test_Feature_Data가 만듭니다.

3. Input_Data와 Test_Feature_Data를 서버에 돌리게 되면 answer.txt와 output.txt가 생성됩니다.
--------------------------------------------------

< 성능 평가 코드>

1. 해당 answer.txt와 output.txt파일을 evaluation.py와 같은 폴더 내 위치하도록 한다. 

2. evaluation.py를 run하면 성능평가가 나오게 됩니다. (매트릭스와 성능이 나오게 됩니다)
